# gurl 开发计划更新

## 项目概述
gurl 是一个类似 wrk 的高性能 HTTP 压测工具，支持 curl 命令解析和多种压测模式。本文档描述了项目的后续开发计划。

## 当前状态 (截至 2025-08-29)

### 已完成的功能
- ✅ 完善单元测试和集成测试
- ✅ 实现批量压测配置文件支持
- ✅ 实现URL模板变量支持
- ✅ 实现基础统计功能
- ✅ 优化 MCP 批量测试功能（支持内联测试定义）

### 部分完成的功能
- ✅ 接口级别TPS统计 (仅实现基础统计，未实现按URL分组的TPS统计)

### 未完成的功能
- ❌ Web UI界面
